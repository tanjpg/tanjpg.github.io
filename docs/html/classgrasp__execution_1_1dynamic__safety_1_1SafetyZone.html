bool grasp_execution::dynamic_safety::SafetyZone::set ( const Option &  option)

Update safety zone option.

Based on the timing specified in option, safety zone limit will be calculated. If option is valid, the following output will be printed in the terminal

zone: | BLIND | EMERGENCY | SLOWDOWN | REPLAN | SAFE
0 {value1, value2, value3, value4}
